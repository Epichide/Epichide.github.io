chan vess segmentation

sign illustration

  • f : denote the given grayscale image, f(x),x is point

  • t : iteration t’s border matrix, binary bool ?, describe as iteration th ‘s edge result, is closed curve mask

  • ε : Heaviside regularization,custom params, = 1

  • x : pixel/position in image,2D

  • φ : level set function, φ(i,j),φi,j : φ value at point(i,j)

  • C : an edge set curve

    $$
    C(t)={x \in \Omega: \varphi(x,t)=0}
    $$

    Boundary C represented as the zero level set of implicit function φ(x, y)[^1]

  • H : Heaviside function, for caculate energy function

    $$
    \begin{aligned}
    H(v) &= \begin{cases}1 & v \geq 0 \
    0 & v<0\end{cases} \
    H(\varphi(x, t)) &= \begin{cases}1 & \varphi(x, t) \geq 0 \
    0 & \varphi(x, t)<0\end{cases}
    \end{aligned}
    $$

  • Hε(t,p) : regularised heavyside function, is matrix , size same to image,∈[0,1]

    for example: if border mark 1, unkown area mark 0, that trans to 1 and 0.5

    $$
    H_{\epsilon}(v)=\frac{1}{2}\left(1+\frac{2}{\pi} \arctan \left(\frac{v}{\epsilon}\right)\right)
    $$

  • operator illustration: $\int_{\Omega} H(\varphi(x)) d x$ : sum for all H matrix value —— np.sum(H), is a scale value

  • cinner couter : the optimal values c1 and c2

    $$
    \begin{aligned}
    c_{1} &=\frac{\int_{\Omega} f(x) H(\varphi(x)) d x}{\int_{\Omega} H(\varphi(x)) d x}, \
    c_{2} &=\frac{\int_{\Omega} f(x)(1-H(\varphi(x))) d x}{\int_{\Omega}(1-H(\varphi(x))) d x} .
    \end{aligned}
    $$

  • u : piecewise-smooth function

    $$
    u(x)= \begin{cases}c_{1} & \text { where } x \text { is inside } C \ c_{2} & \text { where } x \text { is outside } C\end{cases}
    $$

  • The third and fourth terms penalize discrepancy between the piecewise constant model u and the input image f.

    $$
    f_{34}=\lambda_{1} \int_{\text {inside }(C)}\left|f(x)-c_{1}\right|^{2} d x+\lambda_{2} \int_{\text {outside }(C)}\left|f(x)-c_{2}\right|^{2} d x
    $$

  • divergence 散度计算:
    $$
    \operatorname{div}(\nabla I)=I_{x x}+I_{y y}
    $$

  • discretize the curvature 曲率计算 :
    $$
    H=\frac{\left(1+U_{x}^{2}\right) U_{y y}-2 U_{x} U_{y} U_{x y}+\left(1+U_{y}^{2}\right) U_{x x}}{2\left(1+U_{x}^{2}+U_{y}^{2}\right)^{\frac{3}{2}}}
    $$

    $$
    H \approx\left(\begin{array}{ccc}\frac{-1}{16} & \frac{5}{16} & \frac{-1}{16} \ \frac{5}{16} & -1 & \frac{5}{16} \ \frac{-1}{16} & \frac{5}{16} & \frac{-1}{16}\end{array}\right) * U
    $$

$$
H=\operatorname{div}\left(\frac{\nabla \varphi}{|\nabla \varphi|}\right)
$$

  • ∇x^+^ denotes forward diﬀerence in the x dimension : f(x+1)-f(x)

    ∇x^−^ denotes backward diﬀerence : f(x)-f(x-1)

    ∇x^0^ =(∇x^+^ + ∇x^−^)/2 is central diﬀerence : (f(x+1)-f(x-1))/2

    process step

  • φ update function:

    φ value at point(i,j) φi,j

    length penalty μ =0.2,

    area penalty ν = 0,

    time step dt = 0.5,

    convergence tolerance tol = 10e−3

    $$
    A_{i, j}=\frac{\mu}{\sqrt{\eta^{2}+\left(\varphi_{i+1, j}^{n}-\varphi_{i, j}^{n}\right)^{2}+\left(\left(\varphi_{i, j+1}^{n}-\varphi_{i, j-1}^{n+1}\right) / 2\right)^{2}}}, \
    \
    B_{i, j}=\frac{\mu}{\sqrt{\eta^{2}+\left(\left(\varphi_{i+1, j}^{n}-\varphi_{i-1, j}^{n+1}\right) / 2\right)^{2}+\left(\varphi_{i, j}^{n}-\varphi_{i+1, j}^{n}\right)^{2}}} .
    \
    g=(A_{i, j} \varphi_{i+1, j}^{n}+A_{i-1, j} \varphi_{i-1, j}^{n+1}+B_{i, j} \varphi_{i, j+1}^{n}+B_{i, j-1} \varphi_{i, j-1}^{n+1}
    -\nu-\lambda_{1}(f_{i, j}-c_{1})^{2}+\lambda_{2}(f_{i, j}-c_{2})^{2})\
    \
    \varphi_{i, j}^{n+1}\leftarrow \frac{[\varphi_{i, j}^{n}+d t \delta_{\epsilon}(\varphi_{i, j}^{n})g]}{
    {[1+d t \delta_{\epsilon}(\varphi_{i, j})(A_{i, j}+A_{i-1, j}+B_{i, j}+B_{i, j-1})] }}
    $$
